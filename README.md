# 동영상플레이어 구현하기 (2025-12-23)

## 구현 목록

- **웹 기반 비디오 플레이어**: Spring Boot 백엔드와 HTML/CSS/JS 프론트엔드 구성
- **제스처 제어**:
  - **싱글 탭**: 재생/일시정지 토글
  - **더블 탭**: 좌/우 영역 터치 시 -10초/+10초 탐색 및 시각적 피드백
  - **드래그 탐색**: 화면 드래그 시 중앙에 절대 시간(예: 1:23) 표시, 놓으면 해당 위치로 이동
- **커스텀 컨트롤 바**:
  - 재생/일시정지 버튼 (SVG 아이콘 적용)
  - 10초 뒤로/앞으로 버튼 (SVG 아이콘 적용)
  - **전체화면 버튼**: 클릭 또는 `Enter` 키로 전체화면 토글
  - 진행 바 및 시간 표시
  - 마우스 중지 시 3초 후 자동 숨김 기능
- **시청자 통계 시각화**: 진행 바 위에 시청자 유지율 히트맵(그래프) 표시
- **편의 기능**:
  - **썸네일 미리보기**: 진행 바 호버 시 해당 시간대 스냅샷 표시 (쓰로틀링 적용으로 성능 최적화)

## 앞으로 구현해야 할 기능들

1. 음향조절 및 음소거 기능, 아이콘
2. 파일명 표출
3. 재생속도 (0.8~2.0, 0.1단위 조절)
4. 스킵 설정 (버튼, 자동, 끄기)
5. 자동 전체화면 ON/OFF
6. 연속재생 ON/OFF
7. 자막 ON/OFF
8. 자막 캡처 ON/OFF
9. 자막 설정 (크기 설정-디폴트값:15, 위치설정)
10. 자막 싱크 (0.1단위 조절)
