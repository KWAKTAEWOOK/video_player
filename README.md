# 동영상플레이어 구현하기 (2025-12-23)

## 구현 목록

- **웹 기반 비디오 플레이어**: Spring Boot 백엔드와 HTML/CSS/JS 프론트엔드 구성
- **제스처 제어**:
  - **싱글 탭**: 재생/일시정지 토글
  - **더블 탭**: 좌/우 영역 터치 시 -10초/+10초 탐색 및 시각적 피드백
  - **드래그 탐색**: 화면 드래그 시 중앙에 절대 시간(예: 1:23) 표시, 놓으면 해당 위치로 이동
- **커스텀 컨트롤 바**:
  - 재생/일시정지 버튼 (SVG 아이콘 적용)
  - 10초 뒤로/앞으로 버튼 (SVG 아이콘 적용)
  - 진행 바 및 시간 표시
  - 마우스 중지 시 3초 후 자동 숨김 기능
- **시청자 통계 시각화**: 진행 바 위에 시청자 유지율 히트맵(그래프) 표시

## 앞으로 구현해야 할 기능들

1. 재생바 옮길때 썸네일 보여주기
2. 전체화면 기능 및 아이콘
3. 음향조절 및 음소거 기능, 아이콘
4. 파일명 표출
5. 재생속도 (0.8~2.0, 0.1단위 조절)
6. 스킵 설정 (버튼, 자동, 끄기)
7. 자동 전체화면 ON/OFF
8. 연속재생 ON/OFF
9. 자막 ON/OFF
10. 자막 캡처 ON/OFF
11. 자막 설정 (크기 설정-디폴트값:15, 위치설정)
12. 자막 싱크 (0.1단위 조절)
